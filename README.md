### Note Taker

Note Taker is a responsive, dynamic web app that allows users to create and delete custom notes.

### Technologies used
* Heroku
* MySQL
* Javascript
* jQuery

### Installation
- Run npm install.

### How to use?
1. Create note in the input field.
2. Add another note or delete.
